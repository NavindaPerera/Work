1. What is the purpose of _LINE_ constant?
    * _LINE_ − The current line number of the file.

2. What is PHP?
    * PHP is a recursive acronym for "PHP: Hypertext Preprocessor". 
    PHP is a server side scripting language that is embedded in HTML. 
    It is used to manage dynamic content, databases, session tracking, even build entire e-commerce sites.

3. What is the difference between include() Function and require() Function?
    * If there is any problem in loading a file then the require() function generates a fatal error and halt the execution of 
    the script whereas include() function generates a warning but the script will continue execution.

4. What is the purpose of $GLOBALS variable in PHP?
    * $GLOBALS − Contains a reference to every variable which is currently available within the global scope of the script. 
    The keys of this array are the names of the global variables.
